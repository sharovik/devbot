# Example of event with scenario
This is a basic example of the event, which supports the scenario.

## Installation guide
To install it please run 
``` 
make build-installation-script && scripts/install/run --event_alias=examplescenario
```

## Usage
Write in PM or tag the bot user with this message
```
write a message
```
Then you will receive the questions from the bot, to which you need to answer.
As the successful result, the bot will send the message to expected channel.
